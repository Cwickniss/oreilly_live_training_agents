## Install

Install using [pyenv](https://github.com/pyenv/pyenv).

```sh
pyenv local
python -m venv .venv
. .venv/bin/activate
pip install -r requirements/requirements.txt
```

Or install with system Python.

```sh
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements/requirements.txt
```
